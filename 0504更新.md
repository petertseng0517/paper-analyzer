啟動後端之後，要啟動前端，前端為使用者會看到的頁面
 
前端啟動如下
 
開新的 terminal
進到前端資料夾：
cd C:\Users\User\Desktop\taica_project\ollama\ollama\frontend
 
啟動前端伺服器:
npm run dev
 
啟動成功後，前端的terminal會顯示類似：
Local: http://localhost:5174/
或 http://localhost:5175/
 
點這個網址就會開啟前端頁面
